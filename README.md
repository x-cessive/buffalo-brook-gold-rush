# Buffalo Brook Gold Rush - Gold Panning Game

## Project Structure
- `scenes/` - Game scenes (menu, main game, etc.)
- `scripts/` - GDScript files for game logic
- `assets/` - Game assets (sprites, sounds, etc.)
  - `sprites/` - Image files
    - `gui/` - UI elements
    - `tools/` - Tool sprites
  - `sounds/` - Audio files
  - `music/` - Music files
- `export_presets.cfg` - Export configuration
- `project.godot` - Main project file

## Game Features
- Gold panning mechanics with realistic sediment separation
- Multiple tools with durability and upgrade systems
- Dynamic economy with fluctuating gold prices
- Performance monitoring with FPS logging
- Error detection and reporting
- Cross-platform compatibility (Linux, Windows, Android)

## Controls
- WASD/Arrow Keys: Move
- E: Interact/Pan for gold
- ESC: Pause game

## Installation
1. Install Godot 4+
2. Open project.godot in Godot editor
3. Run the project

## Export
Use Godot's export system with the provided export_presets.cfg
